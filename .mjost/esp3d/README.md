# README

## Wiring

AUX UART3 pins from the TriGorilla connected to NODEMCU ESP8266:

```
  TG    ->    ESP
-------------------
- VCC   ->    Vin
- GND   ->    GND
- TX    ->    RX
- RX    ->    TX

```
